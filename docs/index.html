post /exportAgentTables.php
Export Agent Table Data (doExportAgentTables)
Exports agent data in the form of a csv depending on the role of the user. If the user is an Agent then only individual agent data is exported, otherwise if the user is a Admin then all table data will be exported.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

exportAgentTablesItem exportAgentTables (optional)
Body Parameter — Necessary Parameters for Exporting Agent Tables

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

CSV File

404

URL Not Found

500

Likely MYSQL error
